Version 1.1

興味を持ってくれたみなさん、ありがとうございます！好意的なレビューや機能のアイデア、バグレポート、それからaTimeLoggerのことを友達に教えていただいたこと、本当にありがとうございました！みなさんからいただくフィードバックが、いつも私たちを前進させる力になります。
ぜひTwitterとFacebookをフォローしていただき、いつでもニュースとイベントの最新情報を手に入れてください！
アップデートの詳細はアプリのブログをご覧ください（英語） www.aloggers.com

新しい機能：
★翻訳：韓国語、中国語（繁体字、簡体字）
★「一日」のカレンダー表示
★パスコードロック
★好きな色を選択可能に
★「期間の編集」で「現在」ボタンを追加
★開始日の時間を「スマート選択」→自動的に前の記録の終了日の時間を選択するようになりました
★画面の自動ロックを「オフ」にすることが可能に

改善した機能：
✓ 円グラフの凡例に時間の詳細を追加
✓ iPhoneをシャッフルすると円グラフの表示が変更可能に → グループ全体、カテゴリー別、両方を表示
✓ 「記録画面」でグループを表示せずに、カテゴリを表示することが可能に
✓ 位置情報サービスの改善

バクの修正：
✝ 旧aTimeLoggerのバックアップを復元する時に、実行中のログがある場合のバグを修正

Version 1.2
Thanks to everyone for your suggestions and positive reviews! It took us a long time to make this new version but number of changes is impressive. Below is just list of what's new. You can read more and find out how to use it in our application blog - www.aloggers.com 

Also don't forget about our Facebook page and Twitter 

New features: 
- Goals 
- History list grouped by activity types 
- Twitter support 
- Archiving activity types 
- 2 new themes 

Improvements: 
- add 'Today' button to Day/Week/Month selector 
- decimal CSV duration 
- minimal duration feature - activities with duration less than minimal value are not saved 
- percents and total added to report 
- select/deselect all button when select multiple activity types 
- added 'Move to' button for grouping 
- Swedish translation 

Bugs fixed: 
- duration bug 
- am/pm in calendar view 
- html report in non-English
