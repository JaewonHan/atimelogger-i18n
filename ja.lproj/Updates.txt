Version 1.3.1

New features: 
- application is now Universal - it looks much better now on the iPad. 
- 'Remind Me' feature will ask what are you doing every N minutes. If you often forget to switch activities, it's a valuable feature for you. Access it in 'Settings' -> 'Advanced'. 
- 'Bulk Delete' feature. Use this feature if you want to delete a large number of historical activity records. 

Improvements: 
- CSV report includes year in dates if needed (see 'Settings' -> 'Reports') 
- added CSV duration format with better precision 
- improved naming of backups 
- backup can be deleted with a swipe in the backup list 
- save activity types selection now available in reports 
- icons and fonts have better quality on non-retina displays 

Bug fixes: 
- fixed wrong countdown when multiple activities were shown
